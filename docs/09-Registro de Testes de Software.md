Todos os testes realizados nesta etapa foram concluídos com sucesso, garantindo que a tela inicial do sistema esteja funcionando perfeitamente e atendendo às necessidades do usuário.

Todos os testes foram concluídos com sucesso, garantindo a precisão e confiabilidade desses documentos.

## Plano de Teste 7 - Exames Pendentes

A funcionalidade de consulta de exames pendentes permite ao usuário pesquisar se existem exames pendentes para um cliente específico. Ao inserir o nome do cliente no sistema, é possível verificar se há algum exame aguardando realização. Essa ferramenta agiliza o processo de acompanhamento e gerenciamento de exames, proporcionando informações atualizadas sobre o status de cada solicitação. Todos os testes relacionados a essa funcionalidade foram concluídos com sucesso, assegurando a confiabilidade e eficiência do sistema de consulta de exames pendentes.

## Plano de Teste 8 - Verificação em Duas Etapas

A validação em duas etapas é implementada no sistema para garantir a segurança dos dados. Nesse processo, dois profissionais inserem seus dados separadamente e, em seguida, ocorre uma comparação para confirmar a consistência das informações fornecidas. Essa abordagem aumenta a confiabilidade dos dados inseridos, evitando erros ou discrepâncias. Os testes realizados confirmaram o 
funcionamento adequado e efetivo da validação em duas etapas no conforto dos dados.
